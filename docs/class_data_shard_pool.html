Detailed Description

DataShardPool manages a pool of DataShard objects for efficient memory usage. It pre-allocates shards and reuses them instead of creating/destroying them. This reduces memory fragmentation and improves performance by avoiding frequent allocations.

The pool is initialized with a configurable size and can grow dynamically if needed. When shards are no longer needed, they are returned to the pool instead of being destroyed.
